# Think & Decide Commands
Commands for structured reasoning and decision-making.

/CHOOSE [A vs B] — Help decide. Weight real factors. Give clear recommendation.
/REALPC [decision] — Honest pros and cons. Include uncomfortable truths.
/BLINDSPOT [situation] — What am I not seeing? What blind spots to check?
/STEELMAN [view] — Strongest possible case for this view. Assume brilliance.
/DEVIL [idea] — Devil's advocate. Strongest argument against this idea.
/SKEPTIC [plan] — Maximum skepticism review. Where are the holes?
/STEPS [problem] — Step-by-step problem solving. Show reasoning at each step.
/RIPPLE [decision] — Second and third-order consequences. What happens downstream?
/PREMORTEM [plan] — It failed 6 months later. What went wrong? Work backward.
/MINTEST [project] — Smallest possible test to validate this idea.
