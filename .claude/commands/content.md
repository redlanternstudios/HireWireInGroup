# Content & Social Commands
Commands for creating content across platforms.

/HOOK [topic] — 5 scroll-stopping hooks. First line only. Make people stop.
/CAPTION [photo/topic] — Short, specific caption. Not generic. Not hashtag soup.
/THREAD [idea] — 5-tweet thread. Hook → value → value → value → CTA.
/CARO [topic] — 5-slide carousel outline. Each slide has one clear point.
/REPURPOSE [post] — Turn this post into 3 formats for 3 different platforms.
/CTA [topic] — 3 call-to-action lines. Clear, specific, low-friction.
/BIO [details + vibe] — Short bio. Platform-appropriate.
/SUBJECT [email/post] — 5 subject line options. Range from direct to curiosity.
/CONTRARIAN [topic] — The argument that challenges the consensus.
/SHORTPOST [idea] — 100-word post. One point. Punchy. No filler.
/COMMENT [post] — 3 smart, specific replies. Show real thinking.
