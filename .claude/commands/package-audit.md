# Application Package Audit

Audit generation, document editing, preview, export, package review, and apply handoff. Do not edit files.

Search for:

- application package
- package preview
- documents
- resume
- cover letter
- export
- docx
- txt
- print
- preview
- save
- generated
- accept
- review

Determine:

- where resume generation happens
- where cover letter generation happens
- where preview happens
- whether preview reflects saved DB state or live local state
- whether export reflects preview or editor state
- whether unsaved changes are warned about
- whether the package has one canonical route
- whether duplicate document routes exist
- whether preview is gated by readiness
- whether apply is gated from preview
- whether package acceptance has one action path

Return a clear verdict and P0/P1 fixes.
