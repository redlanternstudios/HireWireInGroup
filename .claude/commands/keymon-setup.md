# Keymon Setup

Use this command to generate or review the setup prompt for Keymon's Claude / VS Code environment.

Goal: give Keymon a self-installing prompt that creates the Claude operating layer, turns it on through `CLAUDE.md` and `MEMORY.md`, and adds helpful health-check commands.

Output:

1. Copy/paste setup prompt
2. Files the prompt will create
3. Commands included
4. Validation steps
5. First command Keymon should run after setup

The canonical copy lives at:

- `KEYMON_CLAUDE_VSCODE_SETUP_PROMPT.md`
