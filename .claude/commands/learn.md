# Learn & Understand Commands
Commands for breaking down complex topics quickly.

/ELI10 [topic] — Explain like I'm 10. No jargon. Real-world analogies only.
/PRIMER [topic] — Quick crash course. What I need for an intelligent conversation.
/MYTHS [topic] — Common misconceptions. List them and correct each one.
/ANALOGY [concept] — Real-world analogy that makes this concept click instantly.
/QUIZ [topic] — 5 quiz questions that test real understanding, not memorization.
/COMPARE [A vs B] — Real differences. When would you use each?
/PROS & CONS [topic] — Both sides. Honest case for and against.
/SUMS [text] — Summarize in 3 bullets. Most important point in each.
/GLOSSARY [topic] — 10 key terms. Plain English definitions only.
/ASKBETTER [topic] — What better questions should I be asking?
/MENTALMODEL [topic] — The mental model that makes everything else make sense.
