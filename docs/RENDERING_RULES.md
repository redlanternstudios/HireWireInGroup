# HireWire Rendering Rules

## Renderer Responsibilities
- Model outputs structured data only.
- Renderer owns all formatting and layout.
- No freestyle document layout by model.
- Resume output must be ATS safe.
- No tables, no multi-column layout.
- Stable headings, consistent section order.
- Bullet and length limits.
- No filler language.

## Output Types
- Resume
- Cover Letter
- Outreach
- Interview Prep

## Section Order (default)
1. Header
2. Summary
3. Experience
4. Projects
5. Education
6. Certifications
7. Skills
8. Achievements

## Output Constraints
- Bullet points: max 6 per section
- Section length: max 1200 chars
- Resume: max 2 pages (ATS safe)
- No unsupported claims or fabricated content
